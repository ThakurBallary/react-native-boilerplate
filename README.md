# React Native Boilerplate

#### Get Started

```
git clone https://github.com/thakurballary/react-native-boilerplate.git 

cd react-native-boilerplate
```

###### Note: Replace all "rnbpts" with your project name

#### Install packages

```
yarn install
```


#### Run on Android

```
yarn android
```


#### Run on iOS

```
cd ios 

pod install 

yarn ios
```
